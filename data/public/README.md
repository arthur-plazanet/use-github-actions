# Content - public

Public content data consumed by arthurplazanet.com and its API.

## Structure

```
content/
└── projects/       # Portfolio projects
    └── {slug}.json
```

## Usage

Fetched via GitHub API by [api.arthurplazanet.com](https://github.com/arthurplazanet-com/api.arthurplazanet.com).

Types are defined in `/types/` and built to the `dist` branch.
